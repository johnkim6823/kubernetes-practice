# Docker 01-basic 실습

## 학습 목표
- Docker 기본 개념 이해
- 이미지와 컨테이너의 차이점 파악
- 기본 Docker 명령어 실습

## 실습 가이드
자세한 실습 내용은 각 폴더의 README.md 파일을 참고하세요.

1. `hello-world/`: Hello World 컨테이너 실행
2. `simple-app/`: 간단한 웹 애플리케이션 실행
3. `exercises/`: 연습문제

## 시작하기
```bash
# Docker 설치 확인
docker --version

# Hello World 실행
docker run hello-world
```
